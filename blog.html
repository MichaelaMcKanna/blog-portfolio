Reverse Words In A String or Array

Today we are going to complete the solution so that it reverses all of the words within a string passed in. Example:
"I love sushi" becomes "sushi love I”

//first define a function that takes a string as a parameter

function reverseWords(str){
//split the string into an array of words

str = str.split(' ');
//define a variable to hold the reversed version of the array of words ititialized to a single space

let reverse = '';
// iterate over the array of words backward

for (let word = str.length -1; word >= 0; word--){
// if the word at the current index isn’t a space then add it to ‘reverse’

// if there is a space then add the space to ‘reverse’

if (str[word] !== ' '){
if(reverse){
reverse += ' '
}
}
//add the word at the iterated index of str to reverse

reverse += str[word]
}
return reverse
}
The more concise version of this method which uses a helper function would be as follows
Function reverseWords(str){
//split turns the str into an array of words, reverse reverses the order of the array and join rejoins the reversed array

return str.split(‘ ’).reverse().join(‘ ’);
}
